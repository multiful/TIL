### 개념: 인공지능 / 머신러닝 / 딥러닝 계층 구조
- **정의:** AI ⊃ ML ⊃ DL 의 포함 관계
- **특징:**
  - **인공지능**: 인간의 지능과 행동을 모방, 데이터 이해·문제 해결, 학습·추론 등 다양한 작업 수행
  - **머신러닝**: AI 하위 분야, 명시적 규칙이 아닌 데이터로 스스로 규칙 학습, 고전 ML은 주로 선형 데이터에서 동작
  - **딥러닝**: ML 하위 분야, 인공 신경망(ANN) 기반, 활성화 함수로 비선형 패턴 학습, 손실 함수와 역전파로 학습

### 개념: 머신러닝의 종류 (학습 신호 기준)
- **정의:** 학습에 사용하는 신호의 형태에 따라 세 가지로 분류
- **특징:**
  - **지도학습(Supervised)**: 데이터-정답 레이블 pair 관계를 학습 (예: 이미지 분류)
  - **비지도학습(Unsupervised)**: 정답 없이 데이터의 구조나 패턴(군집) 학습 (예: 클러스터링)
  - **강화학습(Reinforcement)**: 환경과 상호작용하며 State → Action → Reward 사이클로 학습 (예: AlphaGo, 자율주행)

### 개념: MLP (Multi-Layer Perceptron)
- **정의:** 입력층 · 은닉층 · 출력층으로 구성된 가장 기초적인 딥러닝 모델
- **특징:**
  - 입력값을 가중치(W)와 곱 → 합산 → 활성화 함수 적용 → 반복
  - **가중치(W)**: 층 간 연결 강도 (기울기에 해당)
  - **Bias(b)**: 절편에 해당, 없으면 항상 원점을 지나야 해 모델 표현력 제한
  - **활성화 함수(f)**: 비선형성 부여 → 복잡한 실세계 데이터 처리 가능

### 개념: 순전파와 역전파
- **정의:** 딥러닝 학습의 두 단계 흐름
- **특징:**
  - **순전파(Forward Propagation)**: 입력 데이터를 연산해 최종 출력 계산 → 손실 함수로 손실 값(L) 계산
  - **역전파(Backpropagation)**: 손실 값에 대한 각 파라미터(W)의 기울기 계산 → 손실이 줄어드는 방향으로 파라미터 업데이트

### 개념: 인공지능 학습 프로세스
- **정의:** 데이터 수집부터 모델 배포까지의 표준 파이프라인
- **특징:**
  1. 데이터 수집
  2. 데이터 전처리
  3. 모델 구축
  4. 모델 학습
  5. 모델 평가

## 3. 중요 포인트

### 기억해야 할 내용
- AlexNet(2012)이 GPU 도입 + 오픈소스 문화를 촉발하여 딥러닝 폭발적 성장의 기점이 됨
- Transformer(2017)는 단순 NLP 모델이 아니라 LLM·BERT·GPT 등 현재 AI 핵심 기술 전체의 원류
- 활성화 함수가 없으면 아무리 층을 쌓아도 결국 선형 모델과 동일 → 비선형성이 딥러닝의 핵심

### 헷갈리는 내용
- AI ⊃ ML ⊃ DL 계층 구조: 딥러닝은 머신러닝의 일종이고, 머신러닝은 인공지능의 일종
- Bias(편향)와 통계적 bias를 혼동하지 않도록 주의 → 여기서 bias는 절편(intercept)의 역할
- 순전파는 예측, 역전파는 학습(파라미터 업데이트) — 두 단계가 번갈아 반복되는 것

## 5. 개인 정리

- **새롭게 알게 된 점:**
